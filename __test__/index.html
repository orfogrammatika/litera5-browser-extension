Редакторы для проверки плагина Литера5

Это страничка для ручной проверки плагина Литеры при работе с различными формами текстовых редакторов. Разместите этот файл на каком-нибудь вашем веб-сайте, затем откройте в браузере и проверяйте работу плагина.

Textarea

Базовое текстовое поле для редактирования крупных текстов textarea.

CKEditor 4

CKEditor версии 4 (https://ckeditor.com/ckeditor-4/resources/).

CKEditor 5

CKEditor версии 5 (https://ckeditor.com/ckeditor-5/resources/). Текст на проверку вроде добывается, но не возвращается обратно.

TinyMCE

Редактор TinyMCE (https://www.tiny.cloud/).

ContentEditable

Любой блочный элемент HTML (div, p) с включенным параметром contenteditable="true".
